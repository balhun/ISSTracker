# ISS Tracker

[https://isstrackerhunor.netlify.app/](https://isstrackerhunor.netlify.app/)

## What It Does

- Shows real-time ISS location
- Shows the future orbit
- Displays altitude and speed
- Updates every 5 seconds

## How to Use

1. Clone repo
2. `npm install`
3. `npm start`

## Tech Used

- React
- Leaflet maps
- WheretheISS.at API
